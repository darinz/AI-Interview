# Reference Materials

This folder contains comprehensive reference materials for machine learning interviews and system design, including cheatsheets, study guides, and authoritative resources.

## Primary Resources

### Machine Learning Interview Book
**Author:** Chip Huyen  
**URL:** [https://huyenchip.com/ml-interviews-book/](https://huyenchip.com/ml-interviews-book/)

A comprehensive guide covering:
- **Part I: Overview** - ML roles, interview process, compensation, career progression
- **Part II: Questions** - 200+ knowledge questions covering:
  - Math (Algebra, Calculus, Probability, Statistics)
  - Computer Science (Algorithms, Data Structures)
  - Machine Learning Workflows
  - ML Algorithms (Classical ML, Deep Learning, NLP, Computer Vision)

**Key Features:**
- Questions organized by difficulty level
- Covers both research and production ML roles
- Includes behavioral questions and negotiation tips
- Real-world insights from both candidate and interviewer perspectives

### Machine Learning Systems Design
**Author:** Chip Huyen  
**URL:** [https://huyenchip.com/machine-learning-systems-design/toc.html](https://huyenchip.com/machine-learning-systems-design/toc.html)

Advanced course covering:
- **Research vs Production** - Performance and compute requirements
- **System Design** - Project setup, data pipelines, modeling, serving
- **Case Studies** - Real-world implementations
- **Exercises** - Hands-on practice

**Target Audience:** Senior ML engineers, ML system architects, technical leads

## How to Use This Reference

### For Interview Preparation
1. **Start with the ML Interview Book** - Understand the interview process and question types
2. **Review relevant cheatsheets** - Use as quick reference during study sessions
3. **Practice with Systems Design** - For senior roles, focus on ML system architecture
4. **Use template.md** - Create custom study materials for specific topics

### Study Path Recommendations

#### Junior ML Engineer
- Focus on: ML Interview Book Part II (Math, CS, ML Algorithms)
- Cheatsheets: ML fundamentals, supervised/unsupervised learning
- Practice: Basic algorithm questions, model evaluation

#### Senior ML Engineer
- Focus on: ML Systems Design course, advanced ML concepts
- Cheatsheets: Deep learning, advanced mathematics
- Practice: System design, scalability, production challenges

#### ML Research Scientist
- Focus on: Advanced mathematics, deep learning architectures
- Cheatsheets: All mathematical foundations, cutting-edge algorithms
- Practice: Research methodology, novel problem solving
